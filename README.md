# Cơm Tấm Sài Gòn Xưa - Đến Với Cố Đô Huế

Dự án Landing Page tĩnh phục vụ cho quán "Cơm Tấm Sài Gòn Xưa" đặt tại thành phố Huế, kết hợp hương vị ngọt béo của ẩm thực miền Nam và nét tinh tế, cay nồng của cố đô.

## Yêu cầu

Dự án này sử dụng hoàn toàn **HTML, CSS, JavaScript thuần** (Vanilla) để tối đa hoá tốc độ và dễ dàng chạy trên bất kỳ môi trường nào (khi chạy không yêu cầu Node.js hay bất cứ công cụ build nào).

## Hướng dẫn cài đặt và chạy

1. Do đây là trang HTML tĩnh tĩnh, bạn không cần chạy `npm install` hay `npm start`.
2. Bạn chỉ cần mở file `index.html` bằng bất kỳ trình duyệt nào (Google Chrome, Safari, Firefox).
   - Truy cập vào thư mục của bộ code.
   - Nhấp đúp chuột vào `index.html` hoặc kéo-thả file vào trình duyệt.

*(Nếu bạn muốn chạy qua server để test, bạn có thể dùng lệnh `python3 -m http.server 8000` hoặc Live Server extension trên VSCode)*

## Các tính năng
- **Responsive**: Giao diện hiển thị tốt trên cả Máy tính (Desktop) và Điện thoại di động (Mobile).
- **Thiết kế cực kỳ bắt mắt**: Bố cục Glassmorphism hiện đại, hình ảnh AI sắc nét (Đĩa cơm tấm trên nền sông Hương lãng mạn).
- **Form đặt bàn**: Validations cơ bản và alert thông báo.
